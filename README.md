# Домаћи задатак из Техничке документације

## Задатак

Програм који на основу унете дужине странице једнакостраничног троугла `a` израчунава површину `P`.

### Формула

\[
P = \frac{\sqrt{3}}{4} \cdot a^2
\]

## Алгоритамска шема

```mermaid
flowchart TD
    Start([Почетак])
    Input[/"Унеси дужину странице a"/]
    Calc[/"P = (√3 / 4) * a^2"/]
    Output[/"Прикажи површину P"/]
    End([Крај])

    Start --> Input --> Calc --> Output --> End
## Објекти
| Променљива | Тип    | Опис |
|-----------|--------|------|
| a         | double | унета дужина странице једнакостраничног троугла |
| P         | double | израчуната површина троугла |
